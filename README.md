# www.ocvaluepro.com

Static marketing site for **OC Value Pro** — Orange County, California date-of-death & estate real estate appraisals. A sister site of [brianward.com](https://www.brianward.com).

- Pure static HTML/CSS/JS. No build step.
- Deploy: Cloudflare Pages (framework preset: None, build command: empty, output dir: `/`).
- 40 Orange County city/community pages under `/areas/`.
- Contact/order form posts to `https://www.brianward.com/api/contact`.
